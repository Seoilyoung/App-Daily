1.3 일정 전체 목록 추가
       1) 앱 실행 → 날짜 다른지 비교 → (같다) 추가 작업 없이 화면 출력
                                      → (다르다) 앱에 등록된 날짜로 전체 목록에 일정 추가 [조건]체크박스에 체크가 하나라도 됐을 때 → 체크박스 모두 해제 및 날짜 변경 → 화면 출력
       2) 일정 전체 목록
                레이아웃 구성 - 원노트 참고
                목록 추가 방식 - 1) 앱 실행 참고
                접근 방식 - 1) 좌상단 혹은 우상단에 아이콘 배치
                            2) 우하단 아이콘 추가
                삭제 불가능
                날짜별 달성률 표시 - 1) 달성률 표시
                                     2) 달성한 일정별로 다른 색으로 표시 (투명도를 다르게 준다던가)

        3) 일정 등록 간편화
                 일정 등록할 때 최근검색기록처럼 목록 표시 1) 각 일정 별로 - 목록에서 삭제 가능
                                                           2) 한 날짜를 통째로
